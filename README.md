# 🎵 Lofeee – Music Streaming Flutter App

A modern music streaming application built using **Flutter**, following **MVVM architecture** with **GetX state management**.

---

## 📁 Project Structure

lib/
│
├── model/        # Data models
├── view/         # UI screens
├── viewmodel/    # Business logic (GetX controllers)
├── services/     # API services
├── utils/        # Helpers & constants
